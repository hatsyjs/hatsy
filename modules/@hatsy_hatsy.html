RequestHandler

RequestHandler<TMatters>: (this: void, context: RequestContext<TMatters>) => PromiseLike<void> | void
  • Defined in hatsy/hatsy/src/request-handler.ts:19

Request processing handler signature.

Handler implementations expect a request processing context containing specific processing matters. E.g. the ones for HTTP request processing. The handler may either respond using the provided matters, or delegate to next handler.

The handler may be asynchronous.

typeparam

A type of request processing matters this handler expects.

        Request processing context containing processing matters.

      Returns PromiseLike<void> | void

      Either nothing if the handler completed its work synchronously, or a promise resolved when the handler completed its work asynchronously.

    Builds a request processing handler that delegates request processing to other handlers.
