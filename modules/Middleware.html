Modification<TInput, TExt>: (this: void, context: RequestContext<TInput>) => RequestModification<TInput, TExt> | PromiseLike<RequestModification<TInput, TExt>>
  • Defined in hatsy/hatsy/src/http/middleware.ts:52

A signature of function extracting request modification after middleware execution.

When middleware modifies the request or response objects such function can be used to extract this modification to pass it downstream.
