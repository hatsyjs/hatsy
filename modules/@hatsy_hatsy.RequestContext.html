Namespace RequestContext<TMatters>

Request processing context.

It is passed to request processing handler. It contains request processing matters the handler needs. The latter can either respond, or delegate processing to next handler.

The context instance is immutable. The matters it contain can be modified or even extended when delegating request processing to the next handler by creating another context based on original one.

Type parameters

  • TMatters

    Request processing matters of this context.
