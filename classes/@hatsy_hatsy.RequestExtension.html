Class RequestExtension<TInput, TExt>

Request extension.

This is a request modifier implementation able to construct a request handler that extends the original request processing means.

It also contains method for request modifications chaining.

Type parameters

  • TInput

    A type of request processing means expected by handlers created by this extension.

  • TExt

    A type of request processing means extension applied by this extension.
