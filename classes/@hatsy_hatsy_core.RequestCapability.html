Class RequestCapability<TInput, TExt>

Request processing capability.

This is a request modifier implementation able to provide request processing means for handlers.

Request processing capabilities could be be combined.

Type parameters

  • TInput

    A type of request processing means required in order to apply this capability.

  • TExt

    A type of extension to request processing means this capability applies.
