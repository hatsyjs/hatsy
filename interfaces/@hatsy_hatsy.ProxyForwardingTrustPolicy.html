trustForwarded: boolean | keyof (string | [string, string])[]
  • Defined in hatsy/hatsy/src/util/trusted-forward.ts:23

Whether to trust forwarding records in Forwarded HTTP request header.

Either a boolean flag, or a list of trusted proxies. The latter can be either a string (trusted by attribute), or a tuple consisting of attribute name and value (e.g. ['secret', 'some_secret_hash']).

default

false, which means the Forwarded header won't be parsed.
