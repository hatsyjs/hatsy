Interface RequestModifierRef<TInput, TExt>

Request modifier reference.

The RequestContext.Agent.modifiedBy method can be used to check whether the target modifier is applied to request already.

This is typically used to prevent the modifier from being applied more than once to the same request.

Type parameters

  • TInput

    A type of request processing means the target modifier is able modify.

  • TExt

    A type of extension to request processing means applied by the target modifier.
