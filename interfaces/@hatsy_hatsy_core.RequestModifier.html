    • Defined in hatsy/hatsy/src/core/request-modifier.ts:53

    Updates subsequent request modifications.

    It is called before request modification applied to the same request after this one. It can update the modification to apply.

    When multiple modifications already applied to request, then this method will be called for each of them in the order reverse to their application (i.e. for the most recent modification first).
